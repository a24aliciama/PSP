DU1 - Exercise 13 - Java ProcessBuilder

Create a program in Java that launches a process that verifies the MD5 checksum of a file and prints that checksum to the standard output.

The following URL shows how to do that verification in Windows 11:  https://www.windowsdigitals.com/verify-md5-sha256-checksum-windows-11/

Use the method getInputStream to print the required output.
